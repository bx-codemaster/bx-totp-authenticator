# Installation
Für die Installation benötigen Sie in der Regel ca. 2 Minuten.

## Hinweis
Erstellen Sie vor der Installation ein Backup der Datenbank.

## Inhalt
- Modul im Adminbereich installieren
- Was die Installation in der Datenbank anlegt
- Modul nach der Installation aufrufen

## Modul im Adminbereich installieren
1. Melden Sie sich im Adminbereich an.
2. Gehen Sie zu **Module > System Module**.
3. Wählen Sie das Modul **BX TOTP Authenticator** aus.
4. Klicken Sie auf **Installieren**.
5. Speichern Sie anschließend die Moduleinstellungen.

## Zugriff nach erfolgreicher Installation
Nach der Installation finden Sie die Suite unter:

- **Admin > Hilfsprogramme > BX TOTP Authenticator**

---

# Deinstallation
1. Melden Sie sich im Adminbereich an.
2. Gehen Sie zu **Module > System Module**.
3. Wählen Sie **BX TOTP Authenticator** aus.
4. Klicken Sie auf **Deinstallieren**.

Bei der Deinstallation entfernt das Modul:

- alle zugehörigen Konfigurationseinträge.
- den zugehörigen Konfigurationsbereich.
- das gesetzte Admin-Zugriffsrecht
